# Analysis scripts for the Crm paper

This repository contains scripts used for data processing, analysis, and visualization in the Crm paper.

## Overview

The scripts are organized by sequencing data type and analysis module, including ChIP-seq, RNA-seq, Nanopore sequencing, and scRNA-seq analyses.
